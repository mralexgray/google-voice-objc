Takes an NSString and returns the text between the two specified tokens.

If it failes to find the two tokens, it will return the entire string, unchanged.

Parameters:
textthe text to parse
startthe token that begins the clipping
endthe token that ends the clipping
includeTokenswhether to include the tokens in the returned text or not. Default is NO
Returns:
the clipped text, or the entire string, if clipping failed
